Buttons

Wir kennen drei Arten von Buttons. Primary, Secondary und Link. Die Button-Breite kann variieren. Für Mobile Lösungen ist eine Breite von 100% beispielsweise durchaus denkbar.

Button tags

Link

Optionen

Pro Seite oder Widget soll wen möglich nur ein Primary-Button verwendet werden. Bei weniger wichtigen Funktionen soll der Secondary-Button verwendet werden.

Typography

Die SGKB verwendet für die Typografie vier Varianten von Titel (h1 bis h4), einen Lead- und einen Paragraph-Stil.

Headings
